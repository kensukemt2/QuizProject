1. セキュリティ監査

🔴 Critical Issues (致命的な問題)

CRITICAL SECRET_KEYの脆弱性
SECRET_KEYが'django-insecure-'で始まっており、開発用の自動生成キーが使用されている
影響: セキュリティ機能が攻撃に対して脆弱
場所: quiz_project/settings.py:24
推奨対応: 50文字以上のランダムな文字列を生成し、環境変数で管理

🟠 High Issues (重大な問題)

HIGH DEBUG設定が本番環境で有効化される可能性
DEBUG設定のデフォルト値がTrueになっている
影響: 本番環境でDEBUGがTrueの場合、内部情報が漏洩する
推奨対応: デフォルト値を'False'に変更
HIGH ALLOWED_HOSTSが空の可能性
環境変数が設定されていない場合、ALLOWED_HOSTSが空リストになる
影響: Host Headerインジェクション攻撃のリスク
推奨対応: 本番環境用のデフォルト値を設定
HIGH HTTPS/SSL関連設定の欠如
SECURE_HSTS_SECONDS, SECURE_SSL_REDIRECT, SESSION_COOKIE_SECURE, CSRF_COOKIE_SECUREが設定されていない
影響: 中間者攻撃やセッションハイジャックのリスク
推奨対応: 本番環境で以下を追加
SECURE_HSTS_SECONDS = 31536000 # 1年 SECURE_SSL_REDIRECT = True SESSION_COOKIE_SECURE = True CSRF_COOKIE_SECURE = True SECURE_BROWSER_XSS_FILTER = True SECURE_CONTENT_TYPE_NOSNIFF = True
HIGH Pillowライブラリの欠如
ImageFieldを使用しているがPillowがインストールされていない
影響: 画像アップロード機能が動作しない
推奨対応: pip install Pillowを実行
