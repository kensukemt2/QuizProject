使用方法

CSVファイルの形式:

カテゴリー,質問,選択肢1,選択肢2,選択肢3,選択肢4,選択肢5,選択肢6,正解番号

サンプルデータ:

数学,2+2は?,2,3,4,5,,3
数学,3×3は?,6,9,12,15,,2
英語,appleの意味は?,りんご,みかん,バナナ,ぶどう,,1
地理,日本の首都は?,東京,大阪,名古屋,福岡,京都,神戸,1

注意事項:

  • 選択肢は最低2個、最大6個まで対応
  • 正解番号は1から始まる数字で指定
  • 同じ質問が既に存在する場合、選択肢は上書きされます
  • CSVファイルはUTF-8またはShift_JISエンコーディングに対応
