シンプル形式

最もシンプルな形式です。質問文の後に選択肢を並べ、正解には*を付けます。

質問文 選択肢1 選択肢2* 選択肢3 次の質問文 選択肢A 選択肢B*

JSON形式

構造化されたデータ形式。プログラムで生成したデータに適しています。

[ { "question": "2+2は？", "choices": [ {"text": "2", "correct": false}, {"text": "3", "correct": false}, {"text": "4", "correct": true}, {"text": "5", "correct": false} ] }, { "question": "日本の首都は？", "choices": [ {"text": "大阪", "correct": false}, {"text": "東京", "correct": true}, {"text": "京都", "correct": false} ] } ]
