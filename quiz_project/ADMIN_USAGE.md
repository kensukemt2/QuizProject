# 🎯 クイズ管理システム - 管理画面利用ガイド

```
http://localhost:8000/quiz-admin/
```
- **統合ダッシュボード**付き
- **リアルタイム統計**表示

```
http://localhost:8000/admin/
```

## 🚀 主要機能とアクセス方法

### 1️⃣ **統合フォーム** - 最も効率的！
**URL:** `/quiz-admin/quiz_api/question/unified-add/`

**特徴:**
- 質問と選択肢を**1画面で同時作成**
- リアルタイムバリデーション
- 正解の排他制御
- 最大6個の選択肢対応

**使用場面:** 1問ずつ丁寧に作成したい場合

---

### 2️⃣ **統一CSV** - 大量データに最適！
**インポート:** `/quiz-admin/quiz_api/category/unified-import/`
**エクスポート:** `/quiz-admin/quiz_api/category/unified-export/`

**CSV形式:**
```csv
カテゴリー,質問,選択肢1,選択肢2,選択肢3,選択肢4,正解番号
数学,2+2は?,2,3,4,5,3
英語,catの意味は?,犬,猫,鳥,魚,2
```

**特徴:**
- **プレビュー機能**付き
- 重複検証
- エラーレポート
- UTF-8/Shift_JIS対応

**使用場面:** Excelで問題を準備済み、一括登録したい場合

---

### 3️⃣ **高度な一括作成** - 柔軟性最高！
**URL:** `/quiz-admin/quiz_api/question/advanced-bulk-add/`

**対応形式:**
1. **シンプル形式**
```
2+2は？
2
3
4*
5

3×3は？
6
9*
12
```

2. **JSON形式**
```json
[
  {
    "question": "2+2は？",
    "choices": [
      {"text": "2", "correct": false},
      {"text": "4", "correct": true}
    ]
  }
]
```

3. **YAML形式**
```yaml
- question: "2+2は？"
  choices:
    - text: "2"
      correct: false
    - text: "4"
      correct: true
```

**使用場面:** プログラムで生成したデータ、複雑な構造の問題

---

### 4️⃣ **従来の一括作成** - シンプル！
**URL:** `/quiz-admin/quiz_api/question/bulk-add/`

**特徴:**
- テキストベースの簡単入力
- 既存形式との互換性

**使用場面:** 素早く少数の問題を作成したい場合

## 🔧 管理機能

### **質問管理** (`/quiz-admin/quiz_api/question/`)
- **list_display:** 質問文、カテゴリー、選択肢数、正解設定状況
- **inline編集:** 選択肢を直接編集
- **フィルタ:** カテゴリー別
- **検索:** 質問文で検索

### **選択肢管理** (`/quiz-admin/quiz_api/choice/`)
- **list_display:** 選択肢文、質問、カテゴリー、正解フラグ
- **フィルタ:** 正解/不正解、カテゴリー別
- **最適化:** パフォーマンス向上済み

### **カテゴリー管理** (`/quiz-admin/quiz_api/category/`)
- **list_display:** カテゴリー名、質問数
- **inline編集:** カテゴリー内で質問作成
- **一括操作:** 選択カテゴリーのCSV出力

## 💡 効率的な使い方

### **📈 推奨ワークフロー**

1. **少数の問題作成**
   ```
   統合フォーム → 1問ずつ丁寧に作成
   ```

2. **大量の問題作成**  
   ```
   Excel等で準備 → 統一CSV → 一括インポート
   ```

3. **プログラム生成データ**
   ```
   JSON/YAML生成 → 高度な一括作成 → インポート
   ```

4. **既存データの編集**
   ```
   統一CSV エクスポート → Excel編集 → 再インポート
   ```

### **🎯 ベストプラクティス**

- **プレビュー機能**を活用してエラーを事前確認
- **重複検証**でデータ品質を保証
- **統計ダッシュボード**で進捗を監視
- **バックアップ**としてCSVエクスポートを定期実行

## 🚨 トラブルシューティング

### **よくあるエラー**

1. **正解が設定されていません**
   - 各質問に必ず1つの正解を設定

2. **最低2つの選択肢が必要**
   - 選択肢は最低2個、最大6個

3. **CSVエンコーディングエラー**
   - UTF-8またはShift_JISで保存

4. **重複質問警告**
   - 同一カテゴリー内での質問文重複をチェック

### **パフォーマンス最適化**

- 大量データ処理時は**プレビュー機能**で事前確認
- **統計API**は5分間隔で自動更新
- **list_select_related**でDB負荷軽減済み
